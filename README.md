* Yahoo: 1996-10-08 Hasta 2016-08-24
* Econstats: 1996-10-10 Hasta 2013-12-10 (No se actualiza mas)
* Lanacion.com.ar: 1/3/2005 Hasta 22/10/2013 (No se actualiza mas)
